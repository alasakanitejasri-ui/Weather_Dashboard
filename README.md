# Weather Dashboard

## Overview
This project is developed as part of Thiranex Task 4: Asynchronous JavaScript & RESTful APIs. It fetches real-time weather information from a public REST API using JavaScript Fetch API and async/await.

## Features
- Search weather by city
- Real-time weather information
- Temperature display
- Humidity display
- Wind speed display
- JSON data parsing
- Fetch API
- Async/Await
- Error handling
- Responsive design

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Fetch API
- OpenWeather API
